Bio

I am an enthusiastic, flexible front-end developer seeking entry-level opportunities in software and web design within London and the South-East.
Over the past 8 years, I have worked for design consultancies and a main contractor on ground engineering projects in Hong Kong and UK. Recently, I have transitioned from the construction industry looking to break into the tech sector.

Since December 2021, I completed a 12-week software development training course discussing the development life cycle, gaining proficiency in HTML5, CSS3, JavaScript ES6, Python3, SQL and mySQL Workbench, NoSQL and MongoDB.

Please take a look at some of the projects that have been developed using these technologies!

Tech Stack

HTML5 is the first language that I learned which sparked my interest in programming and coding. Being a consumer of software and the Internet for most of my life, I was amazed by how straightforward it was to instantly observe (or not observe!) the output and the vast array of content you can add to the page layout. It was very much an "Alice Through the Looking Glass" moment! HTML5 is an effective language for learning programming by doing. With it, i was able to appreciate how modern websites are organised in terms of their content and factored in semantic elements to improve search engine optimisation.

CSS is where you can really show off your artistic flair. I got to grips with the syntax, about appropriate use of selectors and inheritance, when to use flexbox or grid stucture layouts and finally adding my animations and styling for a personal touch. The projects which incorporated CSS were minimalistic in design, simple but effective colour themes, with the aim to make the content more accessible. It was necessary to use styling strategically to entice readers and not to detract from the web products and services. I also gained an appreciation of responsive web design and utilised a Bootstrap Framework to expediate front-end web design output. CSS is an important asset for any front-end developer seeking to provide a premium user experience.

With a foundation of the web content structure, layout and styling appearance, I applied JavaScript fundamental concepts to implement interactivity and functionality. I learned about the Document Object Model and how to access particular html elements using selector queries. Managing and manipulating data required understanding data types and which data structures were best suited for the task. With the information stored, the data values could be rearranged using in-built methods, or constructing functions myself. Comparison and logical operations were performed to render a desired output or response.

I was able to learn why we may choose relational databases and how to create links between one-to-many tables to access specific fields. Working with a small dataset with a predefined structure, I created relations between the tables and learned how to run basic queries. I experimented with the graphical user interface and command line within mySQL WorkBench. Developing database management skills has proven to me its value for organising client and user input data from the front end.

MongoDB is a form of NoSQL database management system storing data as key or field value pairs within documents and those documents within collections as opposed to a table structure. The document file format is an extension of JSON: Binary JSON and allows for efficient traversal over the database. Working with a larger dataset, I familiarised with the system by performing basic queries within MongoDB's interface.

Learning Python helped to consolidate programming concepts on data types, primitive and complex data structures, manipulation using built-in methods and imported modules. It was evident how versatile the language is, and what was learned was a taster to the many applications that Python can provide. Moreover, the syntax is concise and the community for support is active.

Project Portfolio

The Road Less Travelled

What was the Inspiration?
Combining my interest in travel and experience from a former career, this is a geologically themed fictitious travel website showcasing some destinations that I have been fortunate enough to visit.

What was the objective?
To produce a static front-end landing page, contact details and services page.

What skills were learned?
Project planning using sketches and frames. Carrying out research into applying specific aesthetic details where prior knowledge was lacking. From this project, I improved on my ability to apply HTML and CSS fundamental concepts into a front-end design. I gained an appreciation of the importance of structuring content, selection rules and inheritance, and how to arrange layouts appropriate to the view size.

Rock, Paper, Scissors Mini-Game

What was the objective?
To create an independent project displaying the knowledge of fundamental concepts in JavaScript.

What skills were learned?

Implement arithmetic and comparison operators, conditional statements and use of functions. Validate the expected results by stepping through the program. Present the game with an attractive, responsive user interface.

Mini Notes App

What was the objective?
Partially independent and group-based project requiring students to expand on knowledge of manipulating the Document Object Model, JavaScript Object Notation (JSON) and browser local storage concepts.

What skills were learned?
CSS Bootstrap framework impementation, conversion between JS and JSON, object data structures, constructing for loops, use of functions to create and delete notes, show or hide notes and a search bar functionality.

FilmFlix Database

What was the objective?
Use of Python to create, read, update and delete from a SQLite Database.

What skills were learned?
Methods to manipulate lists to arrange data, handling errors with try and except statements within nested if/elif conditionals, basic commands to open, read and write in Python. After preparing the first version, time was invested in conducting manual unit tests (for every stage the user is required to provide an input) and revise the program appropriately.

Here are some resources that helped with my coding journey. Maybe these suggestions will benefit you:
GeeksforGeeks is a computer science community that provides worked examples, theory and interview questions on various computer language fundamental concepts. GeeksforGeeks also run courses for students or if you prefer, self-paced remote classes. I signed up with G4G and the material helped me with brushing up on JS concepts.
The Net Ninja for me is an excellent source for learning computer languages. Shaun delivers the content in bite-size, manageable chunks and strives to keep the lessons up to date and relevant. It's great for learning when you are short of time or looking for a mini-project to apply the skills being learned. You can access most of the videos on Youtube but I am sure that the Net Ninja would be grateful for a subscription to one of his courses!
If you are looking to generate some dummy data for database manipulation, then I would recommend using Mockaroo. It is easy to follow , has a decent range of data types and allows for logic controlled value generation (using Ruby syntax). At the time of writing, the resource was free to use and can output in CSV, JSON, Excel and SQL format to name a few.
Struggling for first time projects or ideas to showcase your new tech skills? Look no further, Colby's 50 projects For React & the Static Web e-book is packed full of ideas to get you started on your portfolio. It has different levels of complexity depending on what you are comfortable with and a lot of helpful illustrations for planning and structuring content.
There are many computer programming quiz/challenges out there. Here's one with a christmas theme: Advent of Code is an annual competition in December that releases two puzzles per day in the run up to Christmas. These puzzles really test your coding skills, don't get discouraged by the complexity- they are challenging after all (your googling skills will improve no doubt!)
Advent of Code can be completed with any programming language and you may find the puzzles do help to solve practical data manipulation problems.
